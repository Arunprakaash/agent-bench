## POST /api/agents

Create an agent.

```bash
curl -X POST http://localhost:8000/api/agents \
  -H "Authorization: Bearer $BENCH_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Agent",
    "provider_type": "rest_api",
    "connection_config": {
      "endpoint": "https://your-app.example.com/bench/run",
      "timeout_ms": 30000,
      "headers": { "Authorization": "Bearer YOUR_TOKEN" }
    },
    "default_llm_model": "gpt-4o-mini",
    "default_judge_model": "gpt-4o-mini"
  }'
```

## POST /api/agents/{id}/test-connection

Test connectivity to the agent's endpoint.

```bash
curl -X POST http://localhost:8000/api/agents/$AGENT_ID/test-connection \
  -H "Authorization: Bearer $BENCH_TOKEN"
```

**Response**
```json
{ "success": true, "message": "Connection successful" }
```
